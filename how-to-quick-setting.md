## How to set up a quick development environment
1. Run svelte template
```shell
npx degit sveltejs/template <project_name>
```

2. Move to project
```shell
cd <project_name>
# To use typescript
# node scripts/setupTypescript.js
```

3. Install dependencies
```shell
npm install
```

4. Start project (test)
```shell
npm run dev
```
